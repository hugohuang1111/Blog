## 前言

npm 代理, 镜像设置

## 镜像

Command:
```cmd
npm config set registry https://registry.npm.taobao.org
```
设置为淘宝镜像


```cmd
npm config get registry
```
查看当前的 registry


```cmd
npm config set electron_mirror https://npm.taobao.org/mirrors/electron/
```
设置 electron 的镜像
